Social Opt-In Pop-Up force visitors to optin using two clicks Social Connect that provides real visitor eMail address used for list subscription.

== Description ==

OptSocial pop-up is a lightweight Social Opt-In plugin specifically designed to convert visitors into leads. Triggered by timer on all site or just on selected pages.

Get real eMail from your visitors and build high-quality list with OptSocial plugin. This plugin locks your site or just selected pages with pop-up and visitors can close it by “Continue with Facebook” and/or “Continue with Google” button(s) giving you permission to grab their real email address and add it to the list. You have full control on when, where and how pop-up will appear. This is just two clicks action and visitors don’t have to enter their information to subscribe to your list, all happens automatically after visitor gives their permission their email will be added to the list. No more lost subscribers because of fake emails, typos, lazy visitors, etc.. only real confirmed emails that people use for their social profiles.

OptSocial pop-up is an easy solution to covert your visitors traffic into leads that can be used further for email marketing.

== Installation ==

= Minimum Requirements =

 * WordPress 3.8+
 * PHP version 5.2.4+
 * MySQL version 5.0+

**This section describes how to install the plugin and get it working**

= Automatic installation (easiest way) =

To do an automatic install of this plugin, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "opt-social" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".

= Manual installation =

**Uploading in WordPress Dashboard**

1. Download `OptSocial.zip`
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `OptSocial.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard

**Using FTP**

1. Download `OptSocial.zip`
2. Extract the `OptSocial` directory to your computer
3. Upload the `OptSocial` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

The WordPress codex contains [instructions on how to install a WordPress plugin](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Frequently Asked Questions ==

= Does this plugin work with newest WP version and also older versions? =
Yes, this plugin works really fine with WordPress 4.4!
It also works great with WP 4.3 - and also should with WP 4.3 - but we only tested extensively with WP 4.4 and 4.3. So you always should run the latest WordPress version for a lot of reasons.
